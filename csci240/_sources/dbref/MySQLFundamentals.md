# MySQL Fundamentals



## Starting The Server

> The first operation is to start the server.



Use the control panel





## File Locations



**Windows**

C:\xampp\mysql\bin



**Mac**

/Applications/XAMPP/xamppfiles/bin/



**Linux**

/usr/local/mysql/bin/



## Terminal 

Windows Command Line

```bash
mysql -u root -p

-- \! [use dos commands like cls to clear screen]
\! cls
```

Power Shell

```powershell
.\mysql -u root -p
```



Mac

```
open mysql
```
